Cоздать “ToDoList”. Должен быть предусмотрен следующий функционал:
a) Сохранение записок производится в файлы. Один файл - одна записка
b) Заметка имеет: Название, Текст.
c) Приложение должно работать через CLI.
d) Файл называется в соответствии с названием заметки.
e) Заметки можно редактировать, удалять по названию, удалять все сразу
f) В разработке CLI должен быть представлен удобный консольный интерфейс
для взаимодействия с программой. Пользователю должны предоставляться
пронумерованные функции на выбор (1. Список заметок 2. Прочитать
заметку 3) и так далее). На пользовательский ввод программа должна
реагировать в соответствии с выбранной опцией. Должна быть
предусмотрена обработка возможных в ходе выполнения программы
исключений.
